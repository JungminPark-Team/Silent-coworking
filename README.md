# 2026 하계 개별학습 소동아리 모각코 활동 계획서

## 1. 소모임명

**JungminPark Team**

## 2. 활동 목적

방학 기간 동안 팀원들이 각자의 코딩 학습 목표를 설정하고, 정기적인 모각코 활동을 통해 자기주도적 학습 습관을 형성하는 것을 목표로 한다.  
오프라인 모임에서는 집중 코딩과 학습 내용 공유를 진행하며, 모임 외 시간에는 개인별 계획에 따라 자율적으로 학습을 이어간다.

## 3. 활동 주제

토익, 알고리즘, 전공 과제 복습, 개인 프로젝트, 코딩 테스트 준비 등 각자 필요한 주제를 선택하여 학습한다.

## 4. 활동 방식

팀원들은 각자 학습 목표를 설정한 뒤, 오프라인 모임에서 함께 학습을 진행한다.

모임 시간에는 개인별 학습을 중심으로 진행하고, 마지막에 각자의 진행 상황과 어려웠던 부분을 간단히 공유한다.

오프라인 모임 외에는 온라인으로 학습 기록을 공유하며 활동을 이어간다.

## 5. 활동 목표

- 방학 중 꾸준한 코딩 학습 습관 형성
- 개인별 학습 목표 달성
- 전공 기초 및 문제 해결 능력 향상
- 팀원 간 학습 동기 부여
- 최종 학습 기록 및 결과물 정리

## 7. 온라인 자율 활동

오프라인 모임이 없는 주에는 개인별 학습을 자율적으로 진행하고, 학습한 내용이나 진행 상황을 온라인으로 간단히 공유한다.

이를 통해 오프라인 모임 횟수는 최소화하되, 방학 기간 동안 지속적인 학습 흐름을 유지한다.

## 8. 기대 효과

최소한의 오프라인 모임으로도 팀원들이 꾸준히 코딩 학습을 이어갈 수 있다.

또한 각자 목표에 맞는 학습을 진행하면서도 정기적인 점검과 공유를 통해 학습 지연을 줄이고, 코딩 실력과 자기주도 학습 능력을 함께 향상시킬 수 있다.

## 9. 최종 결과물

- 개인별 학습 기록
- 알고리즘 문제 풀이 코드
- 전공 학습 정리 자료
- 개인 프로젝트 결과물 또는 코드 정리본
- 활동 회고 및 결과 보고서
